Declaración de Accesibilidad

En nuestra tienda, nos comprometemos a garantizar que nuestros servicios y productos sean accesibles para todas las personas, incluidas aquellas con discapacidades. Trabajamos continuamente para mejorar la accesibilidad de nuestro sitio web de acuerdo con las normas reconocidas.

Nuestros esfuerzos de accesibilidad

  • Nuestro sitio web está diseñado para cumplir con las pautas de accesibilidad WCAG 2.1 de nivel AA.
  • Nos aseguramos de que las imágenes incluyan texto alternativo para los usuarios que utilizan lectores de pantalla.
  • Trabajamos en mejorar la navegación por teclado y la usabilidad para todas las personas.

Medidas para apoyar la accesibilidad

Hemos implementado las siguientes medidas para garantizar que nuestro sitio sea accesible:

  • Auditorías periódicas de accesibilidad.
  • Entrenamiento continuo de nuestro equipo sobre temas de accesibilidad web.
  • Colaboración con expertos en accesibilidad para identificar áreas de mejora.

Retroalimentación

Nos esforzamos por mejorar continuamente la accesibilidad de nuestro sitio web. Si tienes alguna dificultad para acceder a alguna parte de nuestro sitio o si tienes sugerencias para mejorar la accesibilidad, por favor contáctanos:

Compatibilidad con navegadores y tecnologías de asistencia

Nuestro sitio web está diseñado para ser compatible con las tecnologías de asistencia más comunes, incluidos los lectores de pantalla y la navegación por teclado. Recomendamos utilizar navegadores web actualizados para la mejor experiencia posible.

Actualizaciones

Esta declaración de accesibilidad se actualizó por última vez en octubre de 2024. Continuaremos revisando y mejorando la accesibilidad de nuestro sitio para garantizar que todos los usuarios puedan acceder a nuestro contenido.
